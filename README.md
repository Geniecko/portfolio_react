## Radek Kania - Portfolio

### Technologies used

- React 
- React-Router 
- Styled-Components,
- EmailJS,
- CSSTransition,
- Eslint,
- Prettier
- Figma

### Setup

To run this project, install it locally using npm:

- Run ```npm install``` to install packages
- Run ```npm run build``` to build project or run ```npm run start``` for development
